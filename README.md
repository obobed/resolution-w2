# nyt_gui
this is a very simple gui that fetches the latest popular nyt articles and displays them in a simple layout

# running
1. go to the [releases tab](https://github.com/obobed/nyt_gui/tags) and download the latest version for your architecture and os.
2. extract the archive
3. create a .env file and add your NYT api key, which you can get from [here](https://developer.nytimes.com/) (use .env.example as a reference)
4. now you can run it!
